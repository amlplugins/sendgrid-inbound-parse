# @amlplugins/sendgrid-inbound-parse

Native `@sendgrid/client` SDK re-exported under the `@amlplugins` namespace.

> SendGrid Inbound Parse Webhook — parse inbound email and POST as multipart/form-data to your endpoint.

## Install

```bash
npm install @amlplugins/sendgrid-inbound-parse
```

## Use

```ts
import { /* named exports from @sendgrid/client */ } from "@amlplugins/sendgrid-inbound-parse";
// or
import sdk from "@amlplugins/sendgrid-inbound-parse";
```

Built and published by GitHub Actions on every push to `main`.

Part of the amlplugins family — see https://github.com/amlplugins.
